# GUIA DE INSTALACION:

##Requisitos:
* Tener instalado en tu ordenador mysql y java 11

##Pasos:
* deberias crear una base de datos mysql llamada **ong**
  
* Luego, en el fichero application.properties, tenes que cambiar los datos de
spring.datasource.username y spring.datasource.password por tus credenciales de mysql.
